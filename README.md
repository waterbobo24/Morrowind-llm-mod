# Zdo RPG AI - OpenMW Mod

Integration of Zdo RPG AI system into OpenMW.

![Screenshot](./doc/screenshot.jpg)

## Installation

On MacOS:

```
~/Library/Preferences/openmw/openmw.cfg
```

```ini
data="/Users/zdo/Documents/zdo-rpg-ai-openmw-mod/zdorpgai"
content=zdorpgai.omwscripts
```

Run client from https://github.com/drzdo/zdo-rpg-ai and client will automatically connect to the mod.
